## How to play
**If your OS is Windows:**\
Run the **run.bat** file...
1.  by just double clicking it.
2.  by typing **run.bat** into CMD in the same folder.

**If your OS is Linux:**
1.  Open the Terminal
2.  Type in the command "./run.sh"
